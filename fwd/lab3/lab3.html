Zainy Adventures with JQuerry

JQuerry is a cool tool, look what it can do!

Jquerry is a cool tool which can be used to manipulate the Document Object Model of a webpage.

JQuerry can be used...

to do cool stuff..

like animating a box...

to look faded until...

someone hovers over the box...

with their mouse.

Cool, right?

It also lets you do cool things like make the side bars match the page height automatically everytime the screen size changes. That's how the side bars on this page are designed. And finally, it can let you add text into the html to modify the page.
